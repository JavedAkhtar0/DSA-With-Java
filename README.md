# Java Fundamentals and DSA Practice

![Java](https://img.shields.io/badge/Java-Programming-orange)

## Overview

This repository contains Java programs created while learning programming fundamentals, problem-solving techniques, and introductory Data Structures and Algorithms (DSA).

The repository includes programs covering basic Java concepts, conditional statements, loops, pattern printing, number operations, and beginner-level problem-solving exercises.

---

## Topics Covered

- Variables and Data Types
- Conditional Statements
- Loops
- Functions and Methods
- Number Operations
- Pattern Printing
- Switch Case Programs
- Mathematical Operations
- Basic Problem Solving
- Introduction to Data Structures and Algorithms
